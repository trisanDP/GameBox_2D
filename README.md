Introduction:
This Project contains 5 games which users can play to calculate their cognative ability.
Right now, it uses custom min/max score instead of an collected real world data to calculate users cognative abality. So it doesnt depict the actual stats, but is not complectly inaccurate either.  

## Systems Within The Project:
### 1. Score Save using json.
### 2. Dynamic Level Generation for some games.
### 3. Google Login
### 4. Score and Stat Conversion and Management and more
